hk_sub_

What is this?
Execute a command in response to the time line and the reply of Twitter.You can run the command at a high speed because it corresponds to UserStream.
It is actually running at @hk_sub_.
Main functions
  • update_name [name]
  • update_icon [media]
  • omikuji
  • copy [user]
  • sha1 [string]
Install
If you run hk_sub_.
There are several necessary gem and tools.Run the bundle install After you install the contents of the following list to install the dependent libraries.
  • Ruby (2.0 or later)
  • bundler (gem install bundler)
  • SQLite3 (Debian : apt-get install sqlite3)
