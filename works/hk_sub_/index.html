hk_sub_

これはなに?
Twitterのタイムラインやリプライに反応してコマンドを実行するアプリケーション。ユーザーストリームに対応してるので高速にコマンドを実行出来ます。
@hk_sub_で実際に動いてます。
主な機能
  • update_name [name]
  • update_icon [media]
  • omikuji
  • copy [user]
  • sha1 [string]
インストール
自分でhk_sub_を実行する場合。
いくつか必要なgemとツールがあります。以下のリストの内容をインストールしたらbundle installを実行して依存ライブラリをインストールします。
  • Ruby (2.0以降)
  • bundler (gem install bundler)
  • SQLite3 (Debian : apt-get install sqlite3)
